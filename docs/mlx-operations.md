# MLX 운영 가이드

이 문서는 더 이상 현재 운영 기준 문서가 아니다.

standalone `mlx_lm.server` 실행 경로와 관련 launchd/plist 는 제거했고, 현재 기본 local runtime 은 host macOS의 llama.cpp LFM2 server 다.

현재 운영 절차는 아래 문서를 기준으로 본다.

- [docs/llama-cpp-operations.md](docs/llama-cpp-operations.md)
- [docs/service-operations.md](docs/service-operations.md)

MLX 관련 내용은 migration history 용으로만 취급한다.
